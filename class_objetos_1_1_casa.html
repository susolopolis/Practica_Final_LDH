Método que permite mostrar un objeto del hogar.

En este método se devuelve un vector de cadenas con la información correspondiente a cada objeto presente en la lista actual de objetos en el hogar. Para ello se indica el índice del objeto que se desea mostrar y posteriormente se simula el desplazamiento de posición y se devuelve la información para ese objeto.
