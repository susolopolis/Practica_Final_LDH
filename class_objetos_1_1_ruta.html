void Objetos.Ruta.calcular_pos_actual ( int  rangoX,
int  rangoY 
)
inline

Método que simula y calcula las posiciones X e Y del objeto para esta ruta.

En este método se calculan las posiciones X e Y del objeto en base a un rangoX y rangoY que serán las cotas sobre las que se deseará ejecutar la simulación, de tal modo que se pueda ajustar y reflejar un desplazamiento del objeto más realista. Para aportar cierta naturalidad a la operación, se dota de aleatoriedad a ésta y se distingue entre 4 casos de ejecución: incrementar X y decrementar Y; decrementar X e incrementar Y; decrementar X e Y; incrementar X e Y.

Parameters
rangoXUn entero. El rango en el que se desea incrementar/decrementar el valor de X.
rangoYUn entero. El rango en el que se desea incrementar/decrementar el valor de Y.
